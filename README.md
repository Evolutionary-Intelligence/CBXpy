# What is CBXPy?

```CBXPy``` is a python package for consensus-based particle dynamics, focosing on **optimiation** and **sampling**.

For an ensemble of particles $x = (x_1, \ldots, x_N)$ the basic update step reads

```math
x_i \gets x_i - dt\, (x_i - c(x))
```
